# HSR Frontend

Фронтенд для учебного симулятора управления ресурсами на React + Vite.

## Что уже есть

- главная страница и ввод параметров проекта;
- игровая страница с диаграммой, ресурсами и логикой хода;
- страница результатов с баллами, таймлайном и приоритетными рекомендациями.

## Запуск локально

```bash
npm install
npm run dev
```

## Проверка качества

```bash
npm run lint
npm run build
```

## Основные файлы

- `src/App.jsx` — маршруты и общий каркас приложения;
- `src/pages/ResultPage.jsx` — интерфейс страницы результатов;
- `src/pages/ResultPage.css` — стили страницы результатов;
- `src/data/resultMock.js` — мок-данные для экрана результатов;
- `src/index.css` — глобальные стили и токены дизайна.
