# HSR Frontend - Results Page

Фронтенд для симулятора управления ресурсами ДПИШ РУТ МИИТ.

В текущем состоянии реализован экран `ResultPage` с акцентом на:

- минималистичный UI в палитре, близкой к корпоративной стилистике РУТ;
- компактный UX-поток: итог, метрики, структура балла, ключевые события, рекомендации;
- визуально отличимую типографику (комбинация `Manrope` + `Prata`);
- кастомные inline-иконки без внешних икон-паков.

## Быстрый запуск

```bash
npm install
npm run dev
```

## Проверка качества

```bash
npm run lint
npm run build
```

## Структура новой страницы

- `src/App.jsx` - точка входа приложения, рендер `ResultPage`.
- `src/pages/ResultPage.jsx` - структура экрана и UI-логика.
- `src/pages/ResultPage.css` - стили страницы (hero, KPI, timeline, рекомендации).
- `src/data/resultMock.js` - мок данных для сессии симуляции.
- `src/index.css` - глобальные токены, шрифты и базовые стили.
